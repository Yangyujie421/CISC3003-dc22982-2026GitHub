How to run:

4. Install PHPMailer in this project folder:
   composer require phpmailer/phpmailer
5. Open php/mail_config.php and replace the Gmail address and app password.
6. Visit: http://localhost/CISC3003-FinalExam-Paper02B/index.php
7. Use debug.php if email sending fails.
